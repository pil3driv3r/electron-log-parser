# Electron log parser

Building a tool to parse logs and slice and dice them based on facets.


## Prerequisite

Project is built on node 16.14.2 and uses React 17.0.2, MUI 5.


### `nvm use`

Windows only

### `nvm use 16.14.2`


## Installing dependencies and running the project

### `npm install -g foreman`
### `npm install`
### `npm run start`
